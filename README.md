# 库存管理系统--退货版

《库存管理系统--退货版》差异说明：  
1.在入库管理中增加入库退货、入库退货期间统计，退货时根据入库单号退货；  
2.在出库管理中增加出库退货、出库退货期间统计，退货时根据出库单号退货；  

# 如何运行本系统?

本系统基于宏达数据库信息管理开发平台开发，下载后，双击文件夹中"dbimp.exe(或Hadp.exe)"文件即可完整运行对应的信息管理系统。
